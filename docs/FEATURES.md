  - 카테고리별 필터링
  - 최신순/인기순 정렬
  - Markdown 렌더링

- **게시글 상호작용**
  - 좋아요 기능
  - 댓글 작성
  - 신고 기능

### 1.2 댓글 시스템

- **댓글 관리**
  - 신고 기능
  - 관리자 삭제 기능
